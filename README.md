# 📊 Data Analytics Project

> **An End-to-End Data Analytics Project using Python, SQL, Power BI, and Gamma**

This project demonstrates the complete data analytics lifecycle, from loading raw data to generating business insights through data analysis, SQL querying, interactive dashboards, reporting, and presentation. It showcases practical skills in data analytics using industry-standard tools.

---

# 📌 Project Overview

The objective of this project is to analyze a real-world dataset and transform raw data into actionable business insights. The project follows a structured analytics workflow, including:

* 📥 Loading the dataset using Python
* 🔍 Performing Exploratory Data Analysis (EDA)
* 🧹 Cleaning and preprocessing the data
* 🗄️ Running SQL queries using PostgreSQL, MySQL, or SQL Server
* 📊 Creating an interactive Power BI dashboard
* 📝 Preparing a detailed business report
* 🎯 Designing a professional presentation using Gamma

# ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/gunnk06/Data-Analytics-Project.git
```

### Navigate to the Project

```bash
cd Data-Analytics-Project
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run the Project

1. Open Jupyter Notebook.
2. Execute the notebooks in the following order:

   * Data Loading
   * EDA
   * Data Cleaning
3. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
4. Execute the SQL queries from the `sql` folder.
5. Open the `.pbix` file in **Power BI Desktop**.
6. Review the Business Report and Gamma Presentation.
